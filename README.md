# kubectl-lsdeploy
Python plugin for kubectl that prints our the dependencies between deployments and pods

Tested with
  - Python 3.8.0 on Windows 10 (see Issues)
  - Python 3.5.2 on Ubuntu 16.04.06 LTS (x64)
  - Python modules:
    - kubernetes: 10.0.1
    - PrettyTable: 0.7.2
  - Kubernetes:
    - 1.17 
    - 1.16
